# Design System

This file is a starting point. It should evolve as products are built.

## Visual Feeling

Calm. Human. Spacious. Trustworthy.

The interface should feel more like a quiet room than a dashboard.

## Typography

Use typography that feels warm, clear, and legible.

Headings may carry character.

Body text must prioritize readability.

## Color

Use color sparingly.

Color should guide attention, not decorate everything.

Prefer warm neutrals, deep contrast, and gentle accents.

## Motion

Motion should reassure, not entertain.

Use motion to show listening, progress, presence, and completion.

Avoid motion that distracts from the human moment.

## Components

Every component should earn its place.

Primary actions should be obvious.

Secondary actions should be quiet.

Dangerous actions should be clear and reversible when possible.
