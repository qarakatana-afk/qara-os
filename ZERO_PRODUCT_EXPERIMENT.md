# Zero Product Experiment

Use this experiment when starting from the philosophy instead of a product idea.

## Prompt

Forget all existing product ideas.

Forget Bridge.

Forget Legacy.

Forget translation.

Forget accessibility.

Read only the foundational documents in this repository.

From these principles alone, identify the products that naturally emerge.

For each product, explain:

- What unnecessary friction it removes
- Why it deserves to exist
- What human need it restores
- How it deepens presence
- How it strengthens understanding
- How it earns trust
- What evidence would validate it
- What would make us stop building it

Rank the products by philosophical alignment, not market size.
