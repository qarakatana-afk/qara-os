# Roadmap

## v0.1 — Smallest Trustworthy Version

v0.1 is defined in [MVP_v0.1.md](MVP_v0.1.md).

For v0.1, the acceptance baseline is the successful demonstration that the Kernel can produce transparent, repeatable, and auditable recommendations under governing authority.

### v0.1 Acceptance Baseline

QARA OS v0.1 is considered successful when all outcomes in [MVP_v0.1.md](MVP_v0.1.md) are met, including:

- Complete Kernel execution traces for all prototype recommendations.
- Repeatability threshold met with equivalence criteria defined in Outcome 2.
- Decision Records generated for every recommendation.
