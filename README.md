# Qara OS

**Qara OS** is the operating system for building products from truth, presence, trust, and human connection.

This repository is not an app. It is the foundation every app should inherit from.

Products change. Technology changes. The principles remain.

## Purpose

Qara OS exists to make sure every product begins from the same source:

- Truth before strategy
- Presence before performance
- Trust before engagement
- Simplicity before capability
- Human connection before technology
- Evidence before ego

## How to Use This Repository

Before building any product, read these documents in order:

1. `01_founders_charter/FOUNDERS_CHARTER.md`
2. `02_art_of_presence/ART_OF_PRESENCE.md`
3. `03_bridge/BRIDGE_BIBLE.md`
4. `03_bridge/COMPANION_CONSTITUTION.md`
5. `04_design/UX_PRINCIPLES.md`
6. `04_design/MEMORY_PHILOSOPHY.md`
7. `05_engineering/ENGINEERING_PRINCIPLES.md`
8. `06_genesis/GENESIS_FRAMEWORK.md`
9. `07_ai/AI_MASTER_PROMPT.md`

Then duplicate `08_templates/product_template/` for any new product.

## Core Question

Every product must answer:

> What unnecessary friction can we remove so people have more time for what matters?

## North Star

Technology should disappear behind meaningful human experience.
